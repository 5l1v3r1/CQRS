Cqrs.Azure.BlobStorage.Events.TableStorageEventStoreConnectionStringFactory Class Reference

A factory for getting connection strings and container names for IEventStore<TAuthenticationToken> access. This factory supports reading and writing from separate storage accounts. Specifically you can have as many different storage accounts as you want to configure when writing. This allows for manual mirroring of data while reading from the fastest/closest location possible. Detailed Description

A factory for getting connection strings and container names for IEventStore<TAuthenticationToken> access. This factory supports reading and writing from separate storage accounts. Specifically you can have as many different storage accounts as you want to configure when writing. This allows for manual mirroring of data while reading from the fastest/closest location possible.
